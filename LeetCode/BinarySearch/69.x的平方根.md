## 解题思路
+ 这道题类似9.5的腾讯笔试题目
+ 参考leetcode有3个题解：分别为1、二分查找 2、求x^(1/2)转为求ln的形式 3、牛顿迭代法
+ 主要是多了二分的思路，避免本人继续当一只井底之蛙



## 代码


```
class Solution:
    def mySqrt(self, x: int) -> int:
        l, r, ans = 0, x, -1
        while l <= r:
            mid = (l + r) // 2
            if mid * mid <= x:
                ans = mid
                l = mid + 1
            else:
                r = mid - 1
        return ans 
```
