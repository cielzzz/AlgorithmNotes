## 解题思路

+ 排序法  时间ONlogN
+ 哈希表


## 代码
排序法 时间复杂度ONlogN
```
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        nums.sort()
        mid = nums[len(nums)//2]
        mid_n = sum(i == mid for i in nums)
        return mid if mid_n> len(nums)//2 else -1

```
哈希表
```
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        n = len(nums)
        haspmap = defaultdict(int)
        for i in nums:
            haspmap[i] += 1
            if haspmap[i] > n // 2:
                return i
        return -1
```
