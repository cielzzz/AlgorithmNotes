## 解题思路

主要要注意是从1开始，即将A~Z对应1~26，1对应A而不是0对应A，所以需要n -= 1



## 代码

```
class Solution:
    def convertToTitle(self, columnNumber: int) -> str:
        n = columnNumber
        res = "" #存放结果
        while n:
            n -= 1
            res = chr(ord('A') + n % 26) + res
            n //= 26
        return res
```
