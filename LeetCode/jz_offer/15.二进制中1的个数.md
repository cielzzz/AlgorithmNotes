## 解题思路
 
位运算中
**与运算定义** 
+ 1 & 1 = 1  若n&1 = 0， 则n二进制最右一位为0
+ 1 & 0 = =  若n&1 = 1， 则说明n二进制最右一位为1
本题需要计算的是1的个数，所以可以考虑循环判断：
+ 判断n的最右一位是否为1，根据结果计数；
+ 将n右移一位






## 代码


```
class Solution:
    def hammingWeight(self, n: int) -> int:
        res = 0
        while n:
            res += n & 1 #判断n二进制形式最右一位为1/0
            n >>= 1 #每次右移一位（无符号右移）
        return res
```
